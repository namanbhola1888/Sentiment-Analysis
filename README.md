### Short Video Analytics

#### Project Overview

* A beginner-level project to analyze a video using Heatmap.
* Uses pre-trained models for analysis.

#### Tools and Libraries Used

* **FER Library**: Detects facial emotions from video.
* **Speech Recognition**: Converts speech into text.
* **VADER (NLTK)**: Analyzes positive and negative sentiment from text.

#### Output

* Heat maps showing:

  * Positive and negative facial emotions
  * Positive and negative text sentiments
